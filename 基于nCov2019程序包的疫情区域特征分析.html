nCov2019的来源：

作者：Guangchuang YU（创建者和维护者）
《用于研究COVID-19冠状病毒爆发的开源分析工具》
(发表于《COVID-19 SARS-CoV-2 preprints from medRxiv and bioRxiv》）
https://github.com/GuangchuangYu/nCov2019.

  该程序包用于获取有关冠状病毒爆发的流行病学数据，其中包含详细的实时统计信息，也包含中国详细的城市级数据和世界范围内的国家级数据。数据将从2020年1月13日起每天自动更新。
  数据中的变量包括国家、省、城市、确诊人数、疑似人数、治愈人数、死亡人数等。

随机对照实验（Randomized Controlled Trial）


医学领域进行随机对照实验过程中，研究人员随机将实验样本分为处理组和对照组。

  • 处理组（treatment group）：实施某种医疗或药物干预；

  • 对照组（control group）：不实施干预，其他医疗环境一样。

  • 干预效应：处理组的效果与对照组的效果的差异。

政策评估框架（准自然实验）


政策评估的必要性：经济学家常要评估某政策或时间的经济社会效应，政策可能实施于某些国家或地区,为此，常使用“鲁宾的反事实框架”（Rubin’s counterfactual framework）。

鲁宾的反事实框架（Rubin’s counterfactual framework）：假想该地区未受政策干预将会怎样，并与事实受到干预的实际数据进行对比，二者之差即为“处理效应”(treatment effect)。

常用的解决方法：寻找适当的控制组(control group)，即在各个方面都与受干预地区相似但却未受干预的其他地区，以作为处理组（即受到干预的地区）的反事实替身（counterfactuals）。

准自然试验方法


  • 双重差分法（DID)——构造出反映政策效果的双重差分统计量

  • 倾向得分匹配法(PSM)–找到相似的控制组和干预组

  • 断点回归法(RD)——公共政策因果效应评估

  • 合成控制法(SCM)——构造合成控制地区

  • 回归控制法(RCM)——利用不同个体的截面相关，构造受处理单位的反事实结果

SCM提出背景

合成控制法的使用场景类似于双重差分法，但是不同之处在于干预组只有一个个体。这个个体往往是一个城市、地区或者国家。

  • 比较案例研究(comparative case studies):要考察仅在A市实施的某政策效果，自然会想到以之相近的B市作为控制地区；但B市毕竟与A市不完全相同。或可用其他城市（C市、D市）构成A市的控制组，比较B市、C市、D市与A市在政策实施前后的差别。

  • 缺陷：选择对照地区存在主观随意性(ambiguity),B市、C市、D市与A市也不完全相同。

  • 合成控制法(Synthetic control method):以上述例子来说，虽然无法找到A市的最佳控制地区，但通常可对若干城市进行适当的线性组合，以构造一个更为优秀的“合成控制地区” (synthetic control region)，并将“真实A市”与“合成A市”进行对比，故名“合成控制法”。

SCM的操作步骤


1、筛选合适的控制组，确定合理的预测变量；

2、拟合一个适宜的反事实合成组；

3、对比合成组与处理组的差异以评估政策冲击的影响；

4、安慰剂检验

合成组构造思路

假定只有一个地区（i=1）受政策干预的影响，并且有J个没有政策干预地区（\(i=2，…，J+1\)），\(P_{it}^N\)和\(P_{it}^Y\)分别是地区i在没有政策干预下和有政策干预下的变量取值，政策在\(T_0+1（1≤T_0＜T)\)开始实施，研究样本的地区为\(i=1，…，J+1\)，样本时间为\(t=1，…，T\)。 根据假定，试点地区1的政策效果为：
\(α_{1t}=P_{1t}^Y- P_{1t}^N （t=T_0+1, …, T）\)
其中，\(P_{1t}^Y\)可以被观测，\(P_{1t}^N\)不能被观测。 为了估计\(P_{1t}^N\)\(（t=T_0+1, …, T）\)，利用合成控制法构建一个合成的对照地区：
\(\hat{P}_{1t}^N\) =\(\sum_{i=2}^{j+1}\)\(W_i^t P_{it}^N\)

其中，\(W_i^*\)是地区\(i\)的权重，满足\(W_i^*≥0\)和\(\sum_{i=2}^{j+1}\)\(W_i^* =1\)。

真实加州和合成加州的预测变量


Abadie（2010）证明，在一定的正则条件下，如果合成控制的\(w*\)能完全的复制处理地区的经济特征与干预前的结果变量，则当干预前期数趋向无穷大时，合成控制估计量是渐进无偏的。

反之，如果合成控制\(w^*\)无法很好地复制处理地区的经济特征与干预前的结果变量，则合成控制估计量将是渐近有偏的。因此，在使用合成控制法时，一个重要的模型设定检验就是，考察合成控制的经济特征与干预前的结果变量是否与处理地区足够接近。

安慰剂检验


“安慰剂”（placebo）一词来自医学上的随机实验，比如要检验某种新药的疗效。此时，可将参加实验的人群随机分为两组，其中一组为实验组，服用真药；而另一组为控制组，服用安慰剂（比如，无用的糖丸），并且不让参与者知道自己服用的究竟是真药还是安慰剂，以避免由于主观心理作用而影响实验效果，称为“安慰剂效应”（placebo effect） 安慰剂是一种附加实证检验的思路，并不存在一个具体的特定的操作方法。下面介绍一种常用思路。

以合成控制法为例，通俗一点讲，就是将控制组的每个地区作为处理组，假设这些地区在相同时期也接受了政策干预，按照合成控制法的步骤进行验证，如果政策效果与之前的实验有显著差异，则排除了随机因素影响，认为是政策干预产生的效果。

SCM的优势


  • 作为一种非参数的方法，是对传统的双重查分法的拓展。

  • 通过数据驱动确定权重，减少了主观选择的误差。

  • 通过对多个控制对象加权来模拟目标政策实施前的情况，可以清晰地反映每个控制对象对“反事实”事件的贡献。

SCM的劣势


  • 需要的面板数据比较苛刻，要求干预前的期数达到一定规模；另外，如果政策冲击的效应需要一段时间才会显现（滞后效应），则也要求干预后的期数足够大。

  • 不适用于微观数据。

  • 对比传统回归, 该方法有一个显著缺点:无法得到有意性推断，它无法输出我们常用的P值T值等判断指标, 只能通过安慰性检验等手段侧面手段来佐证。

案例背景介绍

  • 在七十年代恐怖主义活动开始之初（1975），巴斯克地区是西班牙最富有的地区之一，人均国内生产总值（在17个地区中）排名第三位。

  • 在二十世纪九十年代末，经过30年的恐怖和政治冲突，巴斯克地区人均GDP下降到第六位。

  • 在此期间，巴斯克恐怖组织ETA开展的恐怖活动导致近800人死亡。巴斯克地区的企业家和公司已经成为暴力和勒索的具体目标（包括暗杀，抢劫和索要赎金的绑架）。

  • 政治不稳定被认为对经济繁荣有很大的不利影响。

案例具体思路


  • Abadie and Gardeazabal（2003）研究了1975年后恐怖主义冲突对巴斯克地区经济效应的影响

  • 结合其他西班牙地区，使用其线性组合构建了一个“合成巴斯克”地区，并使其经济特征与恐怖主义冲突爆发前的巴斯克地区尽可能相似

  • 比较“合成巴斯克”与真实巴斯克的人均GDP演化

  • Synth程序包：比较案例研究中用于实现合成控制法的R包

  • numDeriv程序包：提供了用于计算准确的数值一阶和二阶导数的方法

  • kernlab程序包：用于分类，回归，聚类，新颖性检测，分位数回归和降维等机器学习方法

  • optimx程序包：扩展的最优化函数，基于导数的算法收敛到全局最小值

案例数据介绍

  • 处理组：受到恐怖活动冲击的巴斯克地区
  • 控制组：其余16个未受到冲击的西班牙地区
  • 潜在假定：恐怖活动仅影响巴斯克地区
  • 结果变量：1955-1997年（冲突发生前后）巴斯克地区和西班牙其他地区的人均GDP： gdpcap
  • 预测变量（13个）：
    • 投资率（投资占总产出的百分比）： invest
    • 产业结构（各产业所占份额）： sec.agriculture、sec.energy、sec.industry、sec.construction、sec.services.venta、sec.services.nonventa
    • 人力资本： school.illit、school.prim、school.med、school.high、school.post.high
    • 人口密度： popdens

案例变量生成

  • 预测变量：

  • X1　：(13x1) 巴斯克地区各预测变量的平均值

  • X0　：(13x16) 其他西班牙地区相应预测变量的平均值

  • 结果变量：

  • Z1　：(10x1) 巴斯克地区在1960-1969年间的人均GDP

  • Z0　：(10x16) 其中每列为相应控制地区在1960-1969 年间的人均GDP

dataprep.out <- dataprep(
  foo = basque,
  predictors = c("school.illit", "school.prim", "school.med",  
  "school.high", "school.post.high", "invest"),
  predictors.op = "mean", time.predictors.prior = 1964:1969,
  special.predictors = list(
    list("gdpcap", 1960:1969 , "mean"),
    list("sec.agriculture", seq(1961, 1969, 2), "mean"),
    list("sec.energy", seq(1961, 1969, 2), "mean"),
    list("sec.industry", seq(1961, 1969, 2), "mean"),
    list("sec.construction", seq(1961, 1969, 2), "mean"),
    list("sec.services.venta", seq(1961, 1969, 2), "mean"),
    list("sec.services.nonventa", seq(1961, 1969, 2), "mean"),
    list("popdens", 1969, "mean") ),
  dependent = "gdpcap",
  unit.variable = "regionno", 
  unit.names.variable = "regionname",
  time.variable = "year",
  treatment.identifier = 17, controls.identifier = c(2:16, 18),
  time.optimize.ssr = 1960:1969, time.plot = 1955:1997)
